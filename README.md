# bbn-t

bbn-t

## Installation

```
$ npm install -g bouzuya/bbn-t
```

## Usage

```
$ export BBN_T_CONSUMER_KEY='...'
$ export BBN_T_CONSUMER_SECRET='...'
$ export BBN_T_ACCESS_TOKEN='...'
$ export BBN_T_ACCESS_TOKEN_SECRET='...'
$ bbn-t 'Hello, Twitter!'
```
